Spider Line 4 Game
IAI_Project.py -> to run the game in the console (input of row and column)
SpiderLine4.py -> To run the game in every mode with the graphic interface
player_player2 -> Graphic mode only player vs player
player_bot.py and player_bot2.py  -> Graphic mode only player vs bot
bot_bot.py ->Graphic mode only bot vs bot
spide4.py -> move function and other functions for the game 
logic.py -> minimax algorithm
tests.py -> code to test different agents with different heuristics
